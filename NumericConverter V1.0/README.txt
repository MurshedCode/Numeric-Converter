



***************Application NumericConverter V1.0 developed uising JDK 1.8 at Eclipse and JUnit 4


Refer to design architecture in analysis and design solution the application includes following files


		RomanConverterApp.java		
		RomanNumeralGenerator.java
		RomanNumeralGeneratorI.java

Test class: 


		RomanNumeralGeneratorTest.java


*****************************************************************************************

The class RomanConverterApp.java run as Java application and produce the output in console.The example outputs are


Entre a number in the range 1 to 3999
9
The converted Roman Numerals: IX

*********************************************

Entre a number in the range 1 to 3999
159
The converted Roman Numerals: CLIX

******************************************


The application can take integer values only at this satge.
